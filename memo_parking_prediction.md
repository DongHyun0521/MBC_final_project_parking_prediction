수/목요일 : 환자 x 1.0

금요일 : 환자 x 1,15

토요일 : 환자 x 1.4



09:00 \~ 10:00: 환자 x 1.2 (오픈런)

10:00 \~ 11:30: 환자 x 1.4 (오전 피크)

11:30 \~ 12:30: 환자 x 0.8 (오전 진료 마감 전)

12:30 \~ 13:30: 환자 x 0.1 (점심시간: 교대진료)

13:30 \~ 15:30: 환자 x 1.3 (오후 피크)

15:30 \~ 17:30: 환자 x 1.0

17:30 \~ 18:30: 환자 x 0.7 (마감 전)
